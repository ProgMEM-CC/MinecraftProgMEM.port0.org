Welcome to ProgMEM-CC Minecraft Server!

Thank you for visiting! To join our Minecraft server, follow the instructions below:

For Minecraft Java Edition:

1. Open Minecraft Java Edition.

2. Click on "Multiplayer".

3. Click "Add Server".

4. In the "Server Address" field, type minecraftprogmem.port0.org.

5. Click "Done" and then select the server from the list to join!

For Minecraft Bedrock Edition:

1. Open Minecraft Bedrock Edition.

2. Click on "Play" and then the "Servers" tab.

3. Click "Add Server".

4. In the "Server Name" field, type any name you like (e.g., "ProgMEM-CC Minecraft Server").

5. In the "Server Address" field, type minecraftprogmem.port0.org.

6. In the "Port" field, enter 19132 (default for Bedrock Edition).

7. Click "Save" and then select the server to join!
